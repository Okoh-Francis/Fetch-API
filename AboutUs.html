Showing you what a GitHub A.P.i Is Used For

An A.P.I is used to let two different apps or systems communicate with each other. API can also be used to get information when needed from another website to the website you are developing

Examples

1. Getting deta from another service

Example: Your website ask GitHob API for user details GitHub sends back the data.

2. sending data to another service

Example: A payment from sends card details to paystack API paystack processes the payment.
